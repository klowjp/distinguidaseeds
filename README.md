# Distinguida Seeds - Website

Sitio web premium para Distinguida Seeds, especialistas en genética botánica y semillas F1 desde 1994 en Bogotá, Colombia.

## Características
- Diseño minimalista de lujo ("Dark Matter aesthetic").
- Animaciones avanzadas basadas en scroll.
- Catálogo de productos interactivo.
- Galería técnica y análisis genético.
- Totalmente responsivo para dispositivos móviles.

## Tecnologías
- HTML5 / CSS3 (Tailwind CSS via CDN).
- JavaScript Vanilla para animaciones.
- Iconify para iconografía técnica.

## Estructura
- `index.html`: Página principal con catálogo y secciones informativas.
- `product.html`: Detalle profundo de cepas genéticas.
- `/images`: Recursos gráficos y logotipos.

## Desarrollo
Para correr el proyecto localmente:
```bash
npm install
npm run dev
```

## Producción
El proyecto está optimizado para carga rápida y unificación cromática en `#0a0a0a`.
